Tour Description

Explore pretty Waiheke Island's beaches, vineyards, and rolling scenery on a full-day, small-group tour that includes tastings at boutique vineyards with the option to visit a Zipline adventures. You'll get an overview and orientation of the island, stopping for great views and sandy beaches, and learn about local wine production as you sip Waiheke vintages.
